This project compares the ethical reasoning capabilities of Mistral and Gemma3 llm using three datasets:

- Commonsense
- Deontology
- Justice

The project evaluates:

- Accuracy
- Response length
- Refusal rate
- Linguistic patterns
- Execution time

Models evaluated:

- Mistral
- Gemma3
